# 🚀 BuildSphere — 60-Day Coding Challenge

**BuildSphere** is a **60-day coding challenge** to build **real-world, portfolio-ready systems**.  
It combines **full-stack development, professional frontend UI and data analytics**, spanning 5 interconnected projects over 8 weeks.  

## 🎯 Challenge Goals
- Build **5 interconnected projects**  
- Showcase **real systems + UI + data analysis**  
- Create **reusable components & templates**  
- Become **portfolio-ready** in 2 months  

## 🛠 Tech Stack
- **Frontend:** React, TypeScript, Tailwind CSS  
- **Backend:** Node.js, Python  
- **Database:** MySQL, SQLite  
- **Analytics:** Python (pandas, matplotlib), Power BI
